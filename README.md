<h1>SnapShop</h1>
<p>SnapShop is an online shopping/eCommerce website made using React.js, Firebase for storing data on the cloud, and ContextAPI for state management and creating functions. Users can sign up and sign in to their accounts, add or remove products from the cart, adjust product quantities, and place orders. User data is saved in the Firebase database, allowing users to view their purchase history anytime.</p>


<h2>Installation and Running Code</h2>
<ul>
    <li>Clone the repository to your local machine.</li>
    <li>Navigate to the root directory of the code.</li>
    <li>Open a command terminal.</li>
    <li>Run the command <code>npm install</code> to install all required dependencies.</li>
    <li>Run the command <code>npm start</code> to start the development server.</li>
    <li>Open the following link in your browser: <a href="http://localhost:3000/">http://localhost:3000/</a>.</li>
</ul>

<h2>Features</h2>
<ul>
    <li>A user can create an account.</li>
    <li>A user can sign in to their account.</li>
    <li>Users can add multiple products to their cart by clicking the "Add to Cart" button.</li>
    <li>Users can remove a specific product or all products at once from the cart.</li>
    <li>Users can increase or decrease the quantity of a product.</li>
    <li>Users can purchase all the products at once by clicking on "Purchase All".</li>
    <li>Users can see their placed order history with the date and time of the order and a list of all the products purchased with the total amount.</li>
    <li>User data will be saved in the database permanently so that they can view their order history anytime.</li>
</ul>
